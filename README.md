# ma-eventos-landing-page

**Sobre o Projeto**

Este projeto é uma Landing Page profissional desenvolvida para a MA Eventos, empresa especializada em painéis de LED e infraestrutura tecnológica para shows. O objetivo foi criar uma interface moderna, intuitiva e totalmente adaptável a diferentes dispositivo.

**Tecnologias e Conceitos Aplicados**

CSS Flexbox: Utilizado para toda a estruturação de layout, garantindo alinhamento dinâmico e distribuição de espaço eficiente.

Arquitetura Semântica: Uso rigoroso de tags HTML5 para otimização de SEO e acessibilidade.

Design Responsivo: Implementação de grids flexíveis que se ajustam automaticamente ao tamanho da tela do usuário.
